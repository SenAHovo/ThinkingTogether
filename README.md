# ThinkingTogether

#### 介绍
智炬五维是一款基于多智能体架构的协同学习对话系统，旨在模拟真实场景下的高质量小组讨论，为用户提供一个具有结构化、多角色、深度互动的智能学习环境。系统内置四个性格与专长各异的AI学伴：理论家擅长梳理知识体系，实践者注重现实应用，质疑者负责挑战观点与逻辑漏洞，组织者则引导讨论节奏并总结关键结论。用户作为第五位成员，可在任意时刻提问、介入讨论、表达观点，从而实现从被动听课到主动思辨的学习方式转变。
